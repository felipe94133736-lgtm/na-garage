# Na Garage Burger's — Site

Site pronto pra publicar. Não precisa de build, servidor nem instalar nada — é um único arquivo `index.html` com tudo dentro (CSS, JavaScript e a logo já embutidos).

## Como publicar no Netlify (o jeito mais rápido)

1. Acesse **https://app.netlify.com/drop**
2. Arraste a pasta inteira (ou só o arquivo `index.html`) pra dentro da página
3. Pronto — o Netlify já te dá um link tipo `nome-aleatorio.netlify.app`

### Se preferir ter uma conta e domínio próprio
1. Crie uma conta grátis em **netlify.com**
2. Clique em **"Add new site" → "Deploy manually"**
3. Arraste este mesmo arquivo/pasta
4. Depois, em **Site settings → Domain management**, você pode:
   - Trocar o nome do subdomínio (ex: `nagarageburger.netlify.app`)
   - Conectar um domínio próprio (ex: `nagarageburger.com.br`), se tiver um

## Se quiser alterar algo depois

Basta abrir o `index.html` em qualquer editor de texto (Notepad, VS Code etc.) e editar diretamente — é tudo HTML/CSS/JS puro. Depois é só arrastar o arquivo atualizado de novo pro Netlify (ou usar "Deploy" → "Drag and drop" na aba do site) que ele substitui a versão publicada.
